# GPU-ROI-ENGINE

GPU profitability calculator suite with embeddable widgets.

## Files

- GPU_ROI_Full.html (2797 lines) - Complete dashboard with grid layout
- GPU_ROI_Ticker.html (2486 lines) - Scrolling ticker display
- GPU_ROI_Ribbon.html (2361 lines) - Horizontal ribbon layout
- GPU_ROI_Ribbon_Ticker.html (2044 lines) - Compact ticker with marquee
- GPU_ROI_Engine.js (1256 lines) - Standalone calculation engine

## Features

- Multi-source ROI calculations (mining, cloud leasing, AI inference)
- Real-time API polling with caching from mining pools and cloud marketplaces
- 4 widget variants: Full dashboard, Ticker, Ribbon, Ribbon_Ticker
- Electricity cost modeling and break-even analysis
- Embeddable via iframe for Wix/Squarespace
- localStorage for persistent settings

## Tech Stack

HTML5, CSS3, Vanilla JavaScript, CSS Grid, Flexbox, localStorage
